# 第一次作业：CAPM 实证检验

## 目标

选取一只 A 股股票和沪深 300 指数，检验 CAPM 模型。

## 要求

1. 使用 `cnstockR` 包获取数据
2. 在 Home 目录下新建一个 `.Rmd` 文件，完成下面的任务并 Knit 为 HTML
3. 绘制回归散点图时使用 `plot()` 函数（base R 绘图）
4. 完成后提交 Home 目录下的 `.Rmd` 文件和 Knit 出的 HTML 文件

## 任务步骤

### 1. 获取数据

用 `cn_get_daily()` 获取一只 A 股股票和沪深 300 指数（000300）2023-01-01 至 2024-12-31 的日线数据，使用前复权（`adjust = 1`）。

建议股票（任选其一即可）：
- 600519（贵州茅台）
- 000858（五粮液）
- 601318（中国平安）
- 000333（美的集团）

### 2. 计算收益率

对股票和指数的收盘价计算对数收益率：`ret = diff(log(close))`。注意对齐日期，去除缺失值。

### 3. 回归分析

以指数收益率为自变量，个股收益率为因变量，做线性回归：`lm(ret_stock ~ ret_index)`。

在 Rmd 文件中展示 `summary(model)` 的完整输出，并回答：
- $\alpha$（截距）是否显著不为零？
- $\beta$ 是多少？该股票相对于市场是激进型（$\beta > 1$）还是防御型（$\beta < 1$）？
- 市场收益能在多大程度上解释该股票的收益波动（看 R²）？

### 4. 绘图

绘制收益率散点图，叠加回归线，标注 $\beta$ 和 $R^2$ 值。仅使用 `plot()` 和 `abline()`。

### 5. 简短讨论

尝试换一只股票或换一个时间段重新跑一遍，$\beta$ 有没有变化？你认为 CAPM 的单一 $\beta$ 能否完整描述股票的风险特征？
