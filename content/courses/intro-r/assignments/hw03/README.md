# 第三次作业：市场弱式有效检验

## 目标

检验 A 股市场指数收益率是否存在自相关，判断市场是否达到弱式有效。

## 要求

1. 使用 `cnstockR` 包获取数据
2. 在 Home 目录下新建一个 `.Rmd` 文件，完成下面的任务并 Knit 为 HTML
3. 绘图使用 `plot()` 和 `acf()` 函数（base R 绘图）
4. 完成后提交 Home 目录下的 `.Rmd` 文件和 Knit 出的 HTML 文件

## 任务步骤

### 1. 获取数据

用 `cn_get_daily()` 获取一只指数的日线数据。

建议指数（任选其一）：
- 000300（沪深 300）
- 000001（上证综指）
- 399001（深证成指）

使用**一个较长的样本期**（至少三年，推荐 2021-01-01 至 2024-12-31），以保证统计检验的有效性。

### 2. 计算收益率

对收盘价计算对数收益率：`ret = diff(log(close))`。去除缺失值。

### 3. 时序图

绘制收益率随时间变化的折线图，添加零线（`abline(h = 0, lty = 2)`）。观察收益率有没有明显的规律或趋势。

### 4. 自相关分析

绘制 ACF 图（`acf()`，建议 `lag.max = 30`）。在 Rmd 文件中回答：
- 有哪些滞后的 ACF 值超出了蓝色虚线（95% 置信区间）？
- 超出置信区间的 ACF 值绝对值是大还是小（例如是否超过 0.1）？

### 5. Ljung-Box 检验

对前 5、10、20 阶做 Ljung-Box 联合检验（`Box.test(..., type = "Ljung-Box")`）。展示每种 `lag` 下的 p 值，判断：

- 在 5% 的显著性水平下，能否拒绝"前 k 阶自相关均为零"的原假设？
- 综合 ACF 图与检验结果，你认为该指数在样本期内是否达到弱式有效？

### 6. 对比分析（选做加分）

将样本期拆分为前半段和后半段（例如 2021-2022 vs 2023-2024），分别做 ACF 和 Ljung-Box 检验。市场效率是否在改善？
