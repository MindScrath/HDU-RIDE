# 第7章 波动模型

在前面几章中，我们的核心目标一直是如何最好地预测时间序列的“均值”（未来是涨还是跌）。在这个过程中，我们一直默认模型的残差（白噪声 $\varepsilon_t$）的方差 $\sigma^2$ 是一个恒定不变的常数。
然而，如果你问一个华尔街的衍生品交易员或风控总监：“你最关心什么？”他们的回答绝不是均值，而是**波动率（Volatility）**。在真实的金融世界里，风险（方差）从来不是静止的，它像海浪一样时而平静，时而波涛汹涌。本章将打破“方差恒定”的经典假设，带你进入现代金融计量学最辉煌的殿堂——波动率建模。

## 7.1 金融数据特征

为什么传统的 ARMA 模型无法完美拟合真实的股票收益率数据？因为高频的金融时间序列（如日收益率、分钟收益率）通常具有以下极其鲜明的特征，这些特征统称为“程式化事实”（Stylized Facts）。

### 7.1.1 波动率聚集现象与尖峰厚尾

**(1) 波动率聚集 (Volatility Clustering)**
数学家曼德勃罗（Mandelbrot）在 1963 年观察到了一个著名现象：“大的价格变动往往跟随着大的变动（无论是上涨还是下跌），而小的价格变动往往跟随着小的变动。”
在图表上，这就表现为市场的“平静期”和“剧烈动荡期”会交替成簇地出现。这意味着：**昨天的方差很大，今天的方差大概率也很大。方差本身存在极其强烈的自相关性！**

**(2) 尖峰厚尾 (Fat Tails / Leptokurtic)**
如果你把真实股票收益率的直方图与标准正态分布叠加在一起，你会发现真实数据的中间部分更高、更尖（尖峰），而两端拖得更长、更厚（厚尾）。这意味着，金融市场发生极端暴涨或暴跌的概率，远远超出了正态分布的理论预期（这就是为什么华尔街经常遭遇所谓的“百年一遇”的黑天鹅事件）。

### 7.1.2 金融风险度量

我们之所以要煞费苦心地去预测未来的方差（波动率），是因为它是现代金融体系的基石：
1. **风险管理 (VaR)**：在计算银行的在险价值（Value at Risk）时，我们需要精确估计明天的波动率，以确保银行有足够的资本储备应对潜在损失。
2. **期权定价**：在著名的 Black-Scholes 期权定价公式中，股票未来的波动率是唯一一个无法直接从市场上观察到、必须由分析师去预测的核心输入变量。

## 7.2 ARCH（自回归条件异方差）模型

为了捕捉波动率聚集现象，1982年，罗伯特·恩格尔（Robert Engle）提出了震惊学术界的 **ARCH 模型（Autoregressive Conditional Heteroskedasticity）**。他因此获得了诺贝尔经济学奖。
ARCH 的核心哲学是：**昨天的剧烈波动（大残差），会导致今天的方差变大。**

### 7.2.1 乘法形式与加法形式

我们在建立收益率模型（如 AR(1)）后，会得到残差序列 $\varepsilon_t$。在 ARCH 模型中，我们不再假设 $\varepsilon_t$ 的方差是常数，而是将其拆解为两部分的**乘法形式**：
$$\varepsilon_t = \sigma_t z_t$$
其中：
- $z_t$ 是严格的白噪声（通常假设服从标准正态分布 $N(0,1)$），它代表了纯粹的、无记忆的系统外部随机冲击。
- $\sigma_t$ 是在时刻 $t$ 的**条件标准差**（即波动率）。注意，它带有下标 $t$，说明波动率是随时间动态变化的！

那么，$\sigma_t$ 究竟是如何变化的呢？恩格尔提出了波动率的**加法形式（条件方差方程）**。对于 ARCH(q) 模型：
$$\sigma_t^2 = \alpha_0 + \alpha_1 \varepsilon_{t-1}^2 + \alpha_2 \varepsilon_{t-2}^2 + \cdots + \alpha_q \varepsilon_{t-q}^2$$
其中，$\alpha_0 > 0$，且所有的 $\alpha_i \ge 0$（以保证算出来的方差永远为正）。

**二者之间的关系与经济学直觉**：
观察加法公式，等式右边是过去 $q$ 期的**残差平方**（$\varepsilon_{t-i}^2$）。如果昨天发生了一个巨大的黑天鹅事件（不管暴涨还是暴跌，导致昨天的 $\varepsilon_{t-1}$ 的绝对值极大），那么 $\varepsilon_{t-1}^2$ 将是一个巨大的正数。
这个巨大的正数乘以系数 $\alpha_1$ 后，会直接导致今天的条件方差 $\sigma_t^2$ 激增。而根据乘法公式 $\varepsilon_t = \sigma_t z_t$，今天的残差波幅也会随之急剧放大。这极其完美地在数学上再现了“波动率聚集”现象！

### 7.2.2 平稳的条件

ARCH(q) 过程的方差随时间剧烈波动，但这只是“条件方差”（已知昨天信息情况下的方差）。对于整个时间序列的“无条件方差”（上帝视角下的平均方差），它必须是平稳的（一个常数），否则整个系统将发散崩溃。

我们对 ARCH(q) 的条件方差方程两边取无条件期望 $E()$：
$$E(\sigma_t^2) = \alpha_0 + \alpha_1 E(\varepsilon_{t-1}^2) + \cdots + \alpha_q E(\varepsilon_{t-q}^2)$$
由于在平稳状态下，长期来看每期的期望方差应该相同，即 $E(\sigma_t^2) = E(\varepsilon_t^2) = Var(\varepsilon_t)$，代入上式得：
$$Var(\varepsilon_t) = \alpha_0 + \alpha_1 Var(\varepsilon_t) + \cdots + \alpha_q Var(\varepsilon_t)$$
移项求解，得到 ARCH 模型的无条件方差为：
$$Var(\varepsilon_t) = \frac{\alpha_0}{1 - \sum_{i=1}^q \alpha_i}$$
要保证方差是一个大于 0 的有限常数，必须满足极其严格的**平稳性条件**：
$$\sum_{i=1}^q \alpha_i < 1$$
如果这些系数之和大于等于 1，任何微小的波动冲击都会在未来被无限放大，最终摧毁整个金融系统。

## 7.3 GARCH（广义自回归条件异方差）模型

在实际应用中，人们发现如果要很好地捕捉漫长的波动记忆，ARCH 模型通常需要设置非常高的阶数 $q$（比如 ARCH(10)）。这会导致需要估计的参数过多，降低模型的精度。
1986年，恩格尔的学生波勒斯列夫（Tim Bollerslev）提出了 **GARCH 模型（Generalized ARCH）**，这也是当今金融界真正大范围普及的工业级标准模型。

### 7.3.1 GARCH 的模型方程

GARCH 模型的核心创新在于：它认为今天的波动率，不仅受到昨天巨大冲击（$\varepsilon_{t-1}^2$）的影响，还受到**昨天本身的波动率预期（$\sigma_{t-1}^2$）的惯性影响**。

**乘法形式**依然保持不变：$\varepsilon_t = \sigma_t z_t$
**加法形式**进化为 GARCH(p,q)：
$$\sigma_t^2 = \alpha_0 + \sum_{i=1}^q \alpha_i \varepsilon_{t-i}^2 + \sum_{j=1}^p \beta_j \sigma_{t-j}^2$$
其中，$\beta_j$ 就是波动率自身的自回归系数。
最常用、最经典的是 **GARCH(1,1)** 模型：
$$\sigma_t^2 = \alpha_0 + \alpha_1 \varepsilon_{t-1}^2 + \beta_1 \sigma_{t-1}^2$$
在这个绝妙的方程中：
- $\alpha_1 \varepsilon_{t-1}^2$ 被称为“ARCH 项”，代表了短期的新闻冲击带来的突发性波动。
- $\beta_1 \sigma_{t-1}^2$ 被称为“GARCH 项”，代表了旧的波动预期的长期衰减惯性。

GARCH(1,1) 极其精简，仅仅通过 3 个参数（$\alpha_0, \alpha_1, \beta_1$）就完美捕捉了几乎所有的波动率聚集特征，平稳条件也顺理成章地变为：$\alpha_1 + \beta_1 < 1$。

### 7.3.2 GARCH 模型与波动率的 ARMA 模型转换

为什么 GARCH 能够用极少的参数代替高阶的 ARCH？因为 GARCH 在本质上就是对方差序列进行的 ARMA 建模！
下面我们将展示这一段极其华丽的代数推导。

我们定义一个新变量 $\nu_t$，它表示真实的残差平方与模型预测的条件方差之间的“意外偏差”：
$$\nu_t = \varepsilon_t^2 - \sigma_t^2$$
显然，$\nu_t$ 是一个均值为 0 的白噪声项（因为在时刻 $t-1$ 看，$\varepsilon_t^2$ 的期望就是 $\sigma_t^2$）。
将 $\sigma_t^2 = \varepsilon_t^2 - \nu_t$ 代入 GARCH(1,1) 的方程：
$$\varepsilon_t^2 - \nu_t = \alpha_0 + \alpha_1 \varepsilon_{t-1}^2 + \beta_1 (\varepsilon_{t-1}^2 - \nu_{t-1})$$
对上述方程进行移项整理，把所有的 $\varepsilon^2$ 放到一起，所有的 $\nu$ 放到一起：
$$\varepsilon_t^2 = \alpha_0 + (\alpha_1 + \beta_1) \varepsilon_{t-1}^2 + \nu_t - \beta_1 \nu_{t-1}$$

请仔细观察最后这个方程！
等式左边是被解释变量 $\varepsilon_t^2$。等式右边包含：
- $\varepsilon_{t-1}^2$ 的滞后项，系数为 $(\alpha_1 + \beta_1)$ $\longrightarrow$ 这是 **AR(1)** 结构！
- $\nu_t$ 及其滞后项 $\nu_{t-1}$，系数为 $-\beta_1$ $\longrightarrow$ 这是 **MA(1)** 结构！

**伟大结论**：**GARCH(1,1) 模型，在本质上完全等价于残差平方序列 $\varepsilon_t^2$ 的 ARMA(1,1) 模型！**
这就解释了为什么 GARCH(1,1) 如此强大——就如同我们在第3章所学，一个极简的 ARMA(1,1) 可以用无穷阶的 MA($\infty$) 来展开；同样地，一个 GARCH(1,1) 其实就等价于一个包含了无穷多期历史冲击的 ARCH($\infty$) 模型。

## 7.4 多元 GARCH 模型

当我们同时管理一个包含苹果、微软和特斯拉的股票投资组合时，我们不仅要预测它们各自的波动率（方差），还要预测它们相互之间的**动态相关性（条件协方差）**。因为在金融危机爆发时，所有股票会罕见地同涨同跌（相关性逼近 1），这会让分散投资的风险对冲策略瞬间失效。

将一元 GARCH 推广到矩阵空间，就诞生了**多元 GARCH 模型 (Multivariate GARCH, MGARCH)**。
设 $E_t$ 是包含多个变量残差的向量矩阵，我们建模的核心从单一的条件方差 $\sigma_t^2$，升维成了**条件协方差矩阵 $H_t$**。

MGARCH 家族有着众多庞杂的分支，其核心痛点在于：矩阵内部的参数数量会随着变量个数呈几何级数爆炸式增长。主流的模型包括：
- **VECH 模型**：最暴力的直接扩展，参数多到几乎无法计算。
- **BEKK 模型**：通过矩阵的二次型乘法（如 $H_t = C^T C + A^T E_{t-1} E_{t-1}^T A + B^T H_{t-1} B$），极其巧妙地保证了生成的协方差矩阵永远是正定的（即方差永远为正）。
- **DCC (Dynamic Conditional Correlation) 模型**：诺奖得主恩格尔在 2002 年的巅峰之作。它极其聪明地将任务拆分为两步：第一步，单独对每个资产算一元 GARCH（极其快）；第二步，专门对它们之间的“动态相关系数”进行建模。DCC 模型是目前华尔街处理超大规模投资组合风控的最核心利器。

### 💻 实战代码：R 语言 GARCH 建模大结局

在 R 语言中，处理波动率模型的无冕之王是 **`rugarch`** 包。我们将完整展示：如何从零开始，对真实股票建立一个带有 ARMA 均值方程和 GARCH 波动方程的联合模型。

```R
# 载入时间序列与高级波动率建模包
library(tidyverse)
library(quantmod) # 用于专业下载金融数据
library(rugarch)  # GARCH 家族终极利器

# --------------------------
# 步骤 1：获取真实数据并计算收益率
# --------------------------
# 使用 quantmod 获取苹果公司 (AAPL) 2020-2023 年的数据
getSymbols("AAPL", from = "2020-01-01", to = "2023-12-31", auto.assign = TRUE)
# 提取收盘价，并计算对数收益率（去除第一天的 NA）
aapl_return <- diff(log(Cl(AAPL)))[-1]

# --------------------------
# 步骤 2：设定联合模型架构 (Specification)
# --------------------------
# 我们需要同时告诉计算机：均值方程长什么样？方差方程长什么样？
garch_spec <- ugarchspec(
  # 1. 均值方程设定：这里我们使用一个简单的 ARMA(1,1) 来拟合收益率本身的走势
  mean.model = list(armaOrder = c(1, 1), include.mean = TRUE),
  
  # 2. 波动方程设定：使用最经典的 GARCH(1,1)
  variance.model = list(model = "sGARCH", garchOrder = c(1, 1)),
  
  # 3. 误差项分布假设：为了更好地拟合“尖峰厚尾”，我们放弃正态分布，改用学生 t 分布
  distribution.model = "std"
)

# --------------------------
# 步骤 3：进行极大似然估计 (MLE) 拟合模型
# --------------------------
# 将设定的蓝图与真实数据结合，进行海量的高级矩阵运算拟合
garch_fit <- ugarchfit(spec = garch_spec, data = aapl_return)

# 查看极其详尽的参数报告
# print(garch_fit)
# 在报告中，重点观察 "Variance Equation" 下的 alpha1 和 beta1 的系数及其显著性。
# 如果 alpha1 + beta1 非常接近 1（例如 0.98），说明这只股票的波动率极其顽固，冲击很难消退。

# --------------------------
# 步骤 4：提取波动率并画图震撼呈现
# --------------------------
# 从拟合好的黑盒中，精准提取出每一天由于历史冲击而计算出的“条件标准差 (波动率)”
conditional_volatility <- sigma(garch_fit)

# 画图：将原本杂乱无章的收益率，转化为极其清晰的、动态起伏的波动风险曲线
plot(conditional_volatility, type = "l", col = "darkred", 
     main = "苹果公司(AAPL) 动态条件波动率曲线", 
     ylab = "条件标准差 (风险度量)")
```
