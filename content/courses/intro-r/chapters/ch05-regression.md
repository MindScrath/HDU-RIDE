# 第5章 时间序列回归模型

在此前的章节中，我们一直局限在“一元”的狭小世界里——试图用茅台过去的收益率去预测茅台未来的收益率。然而，在真实的金融经济系统中，万物皆有广泛的联系。比如，货币发行量（M2）的增速可能会影响股票大盘的走势；美国联邦基金利率的变动会深刻影响全球资本的流动。当我们试图探究**两个或多个时间序列之间的因果和回归关系**时，我们就正式踏入了时间序列回归模型的广阔天地。

## 5.1 静态模型与动态模型

在传统的横截面计量经济学（如探究人的身高与体重的关系）中，回归模型通常是静态的：
$$Y_t = \beta_0 + \beta_1 X_t + \varepsilon_t$$
这表示解释变量 $X$ 在 $t$ 时刻的变化，会立刻、完全地在 $t$ 时刻体现在被解释变量 $Y$ 上。但在时间序列数据中，这种“瞬间完成”的假设往往过于苛刻。例如，央行今天宣布降息（$X$ 变化），股票市场（$Y$）可能在今天会有所反应，但在明天、后天甚至下个月，降息的政策红利和资金流向效应依然在持续发酵。
因此，我们需要引入**动态模型（Dynamic Models）**。

### 5.1.1 分布滞后模型（DLM）

如果在回归方程中，被解释变量 $Y_t$ 不仅受解释变量当期值 $X_t$ 的影响，还受 $X$ 过去若干期滞后值的影响，这就构成了**分布滞后模型（Distributed Lag Model, DLM）**：
$$Y_t = \alpha + \beta_0 X_t + \beta_1 X_{t-1} + \beta_2 X_{t-2} + \cdots + \beta_q X_{t-q} + \varepsilon_t$$

**举例说明**：
假设 $Y_t$ 是某上市公司的季度销售额，$X_t$ 是该季度的广告支出。当期投入的广告 $X_t$ 会立刻刺激当期的销量（由系数 $\beta_0$ 衡量），但精美的广告印象会留在消费者脑海中，因此上个季度甚至上上个季度的广告（$X_{t-1}, X_{t-2}$）依然会对这个季度的销量产生正向的余波作用（由 $\beta_1, \beta_2$ 衡量）。这里的 $\beta_0, \beta_1, \dots$ 构成了广告支出对销售额的**分布滞后效应**。

### 5.1.2 自回归分布滞后模型（ARDL）

虽然 DLM 很好地刻画了 $X$ 的滞后影响，但它忽略了时间序列自身最本质的特征——**惯性**。往往 $Y_t$ 自身也受到其前期表现 $Y_{t-1}$ 的强烈影响。
如果我们把被解释变量 $Y$ 自身的滞后项也加入方程右侧作为解释变量，这就进化成了在宏观经济预测中极为著名的**自回归分布滞后模型（Autoregressive Distributed Lag Model, ARDL）**。
一个包含 $p$ 阶 $Y$ 滞后和 $q$ 阶 $X$ 滞后的 ARDL(p, q) 模型通常写为：
$$Y_t = c + \phi_1 Y_{t-1} + \cdots + \phi_p Y_{t-p} + \beta_0 X_t + \beta_1 X_{t-1} + \cdots + \beta_q X_{t-q} + \varepsilon_t$$

**举例说明**：
假设 $Y_t$ 是当月的通货膨胀率（CPI），$X_t$ 是当月的货币供应量增速。当月的通胀不仅取决于央行当月及过去几个月超发了多少货币（$X_t, X_{t-1}$ 的分布滞后逻辑），还极大地取决于全社会的“通胀预期”——也就是上个月甚至上季度的通胀率 $Y_{t-1}$ 所形成的经济惯性（自回归逻辑）。ARDL 模型完美地将这两股截然不同的力量融合在了一个优美的代数方程中。

## 5.2 干扰分析（Intervention Analysis）

在金融历史的长河中，总有一些突发的“黑天鹅”事件：比如 2008 年的雷曼兄弟破产、2020 年的全球新冠疫情。这些外部的极端冲击（Intervention）会对金融时间序列产生突然且剧烈的结构性影响。

如果在建模时不把这些极端冲击剥离出来，它们会被模型统统当作不明噪音吸收到残差 $\varepsilon_t$ 中，导致残差方差暴增，甚至破坏整个模型的平稳性假设。
为了量化这些事件的真实影响，我们在模型中人为引入**虚拟变量（Dummy Variables）**：
- **脉冲冲击（Pulse Intervention）**：只在事件发生的那个极短时刻起作用。比如某天上市公司意外发布了极差的财报。此时设虚拟变量 $D_t$：仅在事件发生日当天 $D_t = 1$，其余所有历史和未来时间 $D_t = 0$。
- **阶跃冲击（Step Intervention）**：事件发生后，其影响是长期的、永久性改变基本面的。比如国家宣布了新的印花税法规。此时设虚拟变量 $S_t$：在法规实施前 $S_t = 0$，在法规实施当天及未来的所有时间 $S_t = 1$。

将这些高度定制的虚拟变量加入到原本的 ARMA 或 ARDL 模型中，我们就不仅能预测常规的市场波动，还能像剥洋葱一样，精确评估突发政策或历史事件对市场的**绝对量化冲击力度**。

## 5.3 传递函数与格兰杰因果检验

在多元回归模型中，解释变量 $X$ 究竟是如何一步步地将影响“传递”给被解释变量 $Y$ 的？

### 5.3.1 传递函数与脉冲响应

利用我们在第3章中反复使用的神奇武器——滞后算子 $B$，我们可以对复杂的 ARDL 模型进行降维变形。为了直观，我们考察最简单的 ARDL(1, 1) 模型：
$$Y_t = \phi Y_{t-1} + \beta_0 X_t + \beta_1 X_{t-1} + \varepsilon_t$$
提取被解释变量：
$$(1 - \phi B)Y_t = (\beta_0 + \beta_1 B)X_t + \varepsilon_t$$
将等式左侧的自回归多项式 $(1 - \phi B)$ 像除法一样移到右式（数学上的求逆）：
$$Y_t = \frac{\beta_0 + \beta_1 B}{1 - \phi B} X_t + \frac{1}{1 - \phi B} \varepsilon_t$$
$$Y_t = V(B)X_t + N_t$$
在这里，极其重要的核心项 $V(B) = \frac{\beta_0 + \beta_1 B}{1 - \phi B}$ 就被称为系统的 **传递函数（Transfer Function）**。
如果我们将分式 $V(B)$ 用泰勒级数展开，得到的一长串多项式系数 $v_0, v_1, v_2, \dots$ 就是 $X$ 对 $Y$ 的**脉冲响应函数（Impulse Response Function）**。它像慢镜头一样精确描绘了：如果现在给解释变量 $X$ 施加一个单位的冲击，在未来的第 1 期、第 2 期、第 $n$ 期…… 目标变量 $Y$ 将会发生多大的震荡涟漪。

### 5.3.2 格兰杰因果检验 (Granger Causality Test)

当我们把 $X$ 放到方程右边去解释 $Y$ 时，我们内心深处其实潜意识隐含了一个主观假设：是 $X$ 导致了 $Y$。但在统计学上，**“相关”绝对不等于“因果”**（公鸡打鸣和太阳升起在时间上高度相关，但公鸡打鸣绝不是太阳升起的原因）。
诺贝尔经济学奖得主克莱夫·格兰杰（Clive Granger）提出了一种极其精妙、完全基于数据时间先后的因果检验逻辑：
**如果在预测 $Y$ 时，加入变量 $X$ 的历史信息，能够显著地帮助提高对 $Y$ 未来的预测精度（相比于闭门造车只用 $Y$ 自己的历史信息去预测），那么在统计意义上，我们就称“$X$ 是导致 $Y$ 的格兰杰原因”。**

**严谨的检验步骤**：
1. **构建约束回归**（闭门造车模型，只有 $Y$ 的历史）：
   $$Y_t = \alpha_0 + \sum_{i=1}^p \alpha_i Y_{t-i} + u_t$$
2. **构建无约束回归**（加入外部变量 $X$ 的历史）：
   $$Y_t = \alpha_0 + \sum_{i=1}^p \alpha_i Y_{t-i} + \sum_{j=1}^q \beta_j X_{t-j} + \varepsilon_t$$
3. **进行 F 假设检验**，原假设 $H_0$ 为：所有的 $\beta_j$ 都联合为 0（即 $X$ 的加入纯属多余，没有贡献）。
如果在 F 检验中拒绝了原假设（p-value < 0.05），说明 $X$ 的加入确实提供了无可替代的增量预测信息，我们便在统计上正式确立了 $X$ 对 $Y$ 的格兰杰因果关系。

## 5.4 伪回归 (Spurious Regression) 与协整 (Cointegration)

请注意，本节是整个时间序列回归中最危险、也是最容易让初学者“翻车”的绝对雷区！

### 5.4.1 伪回归现象的残酷真相

在经典计量经济学的 OLS 回归中，底层数学定律总是默认数据是平稳的。如果在现实实证中，我们拿两个**完全独立、毫不相干，但都极度非平稳（比如都是在无脑发散的随机游走）**的时间序列去强行跑回归，会发生什么可怕的事情呢？
直觉上，既然它们毫无因果联系，回归算出来的拟合优度 $R^2$ 应该接近于 0，参数的 t 检验应该极其不显著才对。

我们用 R 语言来亲自做一个残酷的计算机蒙特卡洛模拟实验：
```R
library(tidyverse)
set.seed(888)

# 1. 模拟两条完全独立、互不干扰的标准正态白噪声
n <- 500
white_noise_x <- rnorm(n)
white_noise_y <- rnorm(n)

# 2. 累加白噪声，生成两条由于数学构造而毫不相干的非平稳随机游走序列
X <- cumsum(white_noise_x)
Y <- cumsum(white_noise_y)

# 3. 强行用 OLS 做简单的线性回归
fake_model <- lm(Y ~ X)
summary(fake_model)
```
> [!WARNING]
> 如果你在自己的 RStudio 中运行上述代码，你很可能会看到一个极其“华丽”但充满欺骗性的回归报告：回归方程的 $R^2$ 居然可能高达 0.8 以上！解释变量 X 的系数 t 检验 p-value 小于 0.0001，显示其“极其显著”！
> 如果你不懂金融计量学，你可能会立刻拿着这个结果去发表研报，宣称你发现了震惊金融界的隐藏规律。但事实上，这完全是海市蜃楼。仅仅是因为两个非平稳序列都在随时间发散，它们在图形上偶然表现出了同向或反向的虚假趋势重合。这就是时间序列中臭名昭著的**伪回归（Spurious Regression）**陷阱。

**铁律**：绝对不可以对两个非平稳的 $I(1)$ 序列直接进行线性回归！在任何回归开始前，必须先进行第4章讲过的 ADF 平稳性检验！

### 5.4.2 协整检验与 EG 两步法

既然不能对非平稳序列（如绝大多数股票价格天然就是非平稳的 $I(1)$ 序列）直接回归，那我们是不是只能残忍地把它们全部一阶差分成收益率 $I(0)$ 之后再去回归呢？
差分固然可以一劳永逸地解决伪回归陷阱，但差分会带来一个巨大的代价：它将彻底抹杀掉价格序列之间极其宝贵的**“长期绝对价格均衡关系”**。

1987年，计量巨擘恩格尔（Engle）和格兰杰（Granger）横空出世，提出了**协整（Cointegration）**的伟大思想（两人也因此同获诺贝尔经济学奖）：
如果 $X_t$ 是非平稳的 $I(1)$ 序列，$Y_t$ 也是非平稳的 $I(1)$ 序列。但世间存在奇迹，如果能够找到它们之间的某个线性组合：
$$u_t = Y_t - \beta X_t$$
能够使得残差组合序列 $u_t$ 恰好变成**平稳的 $I(0)$** 序列。那么，我们就称 $X_t$ 和 $Y_t$ 之间存在**协整关系**。
协整的经济学意义浪漫而深刻：它意味着这两个变量虽然各自像无头苍蝇一样在金融图表上随机游走，但它们之间似乎存在着一根无形的“弹簧”。长期来看，它们绝不会各自彻底飞散，而是保持着一种强烈的长期动态均衡关系。
*(最经典的通俗比喻：一个醉汉牵着一条狗在公园散步。醉汉的步伐是随机游走的非平稳序列，狗的步伐也是随机游走的非平稳序列，但因为有一根狗绳牵着，醉汉和狗之间的距离（残差）永远是平稳的。)*

**EG 两步法 (Engle-Granger Two-Step Method) 实操步骤**：
1. **第一步（强行长期回归）**：不管三七二十一，直接顶着伪回归的风险，用 OLS 对两个非平稳的 $I(1)$ 序列进行回归：$Y_t = \alpha + \beta X_t + u_t$。计算完毕后，提取出残差序列 $\hat{u}_t$。
2. **第二步（残差平稳性大考）**：对提取出的残差序列 $\hat{u}_t$ 进行无常数项和无趋势项的严格 ADF 检验。
   - 如果残差平稳（ADF 检验拒绝了存在单位根的原假设），证明伟大的协整关系成立！这意味着第一步的强行回归不是伪回归，其回归方程极其珍贵地揭示了变量间的**长期均衡规律**。
   - 如果残差依然不平稳，很遗憾，说明它们之间没有那根“狗绳”引力，第一步的计算就是彻底的伪回归垃圾，应立即废弃。

### 5.4.3 误差修正模型 (ECM)

如果在上一步证实了序列之间存在极其难得的协整关系，我们就不仅拥有了洞察长期的均衡方程，还可以顺势建立捕捉短期剧烈动态的**误差修正模型（Error Correction Model, ECM）**。
根据著名的格兰杰表述定理（Granger Representation Theorem），任何存在协整关系的系统都可以毫无损失地改写为极其优雅的 ECM 形式：
$$\Delta Y_t = \alpha_0 + \sum_{i=1}^p \alpha_i \Delta Y_{t-i} + \sum_{j=0}^q \beta_j \Delta X_{t-j} + \gamma \cdot ecm_{t-1} + \varepsilon_t$$
请仔细端详这个堪称完美的方程结构，其中：
- $\Delta Y, \Delta X$ 是差分后的平稳变量。
- $ecm_{t-1} = Y_{t-1} - (\hat{\alpha} + \hat{\beta}X_{t-1})$，这正是 EG 第一步中提取出的滞后一期的历史残差，它代表了上一时刻系统**偏离长期均衡状态的“误差距离”**。
- **$\gamma$ 被称为至关重要的“误差修正系数”**。在一个合理的、有自我恢复能力的经济系统中，$\gamma$ 必须在统计上显著为**负数**。它生动地代表了：当上一期系统偏离了长期均衡（比如 $Y$ 涨得过高导致误差为正）时，在当期系统会以 $\gamma$ 大小的绝对力度“往回生拉硬拽”，进行反向修正降温，从而确保长期均衡不被彻底打破。

**ECM 模型堪称时间序列回归的巅峰之作**：等式右侧前方的差分项（$\Delta Y, \Delta X$）完美地捕捉了金融市场情绪驱动的**短期波动逻辑**；而等式右侧后方的误差修正项（$\gamma \cdot ecm_{t-1}$）则像定海神针一样，牢牢锁定了核心变量之间的**长期均衡底线**。

### 小结

本章我们终于实现了时间序列分析从孤立的“一元”向复杂的“多元交互回归”的华丽跨越。我们学习了如何用 ARDL 和传递函数刻画历史政策的动态冲击波，如何用虚拟变量评估黑天鹅事件的量化威力，如何用格兰杰因果检验寻找掩盖在数据杂音下的真正经济学动因。最重要的是，我们深刻警惕了可怕的“伪回归”陷阱，并掌握了通过 EG 两步法和误差修正模型（ECM）将长期静态均衡与短期剧烈波动完美融合的现代计量利器。在接下来的第6章中，我们将进一步打破“单向因果”的枷锁，进入所有变量互为因果的向量自回归（VAR）的多体演化宇宙。
