# 市场弱式有效检验

## 有效市场假说（EMH）

Fama（1970）将市场效率分为三个层次：

- **弱式有效**：过去的价格和收益率信息已完全反映在当前价格中，技术分析无法获得超额收益
- **半强式有效**：所有公开信息已反映在价格中
- **强式有效**：所有信息（含内幕信息）均已反映

我们这节课只检验**弱式有效**：今天的收益率是否与昨天的收益率有关？

## 检验方法：自相关

如果市场是弱式有效的，收益率序列应该不存在显著的自相关性。也就是说：

```
ρ(k) = Corr(Rt, Rt-k) ≈ 0    (k = 1, 2, 3, ...)
```

## 获取指数数据

```r
library(cnstockR)

index <- cn_get_daily("000300", start = "2022-01-01", end = "2024-12-31", adjust = 1)
index$ret <- c(NA, diff(log(index$close)))
index <- na.omit(index)
```

## 画收益率时序图

```r
plot(index$date, index$ret, type = "l",
     xlab = "日期", ylab = "对数收益率",
     main = "沪深300 日收益率序列")
abline(h = 0, lty = 2, col = "gray")
```

观察：收益率在零附近波动，有没有明显的趋势或模式？

## 自相关图（ACF）

```r
acf(index$ret, main = "沪深300 收益率自相关图", lag.max = 30)
```

解读 ACF 图：
- 蓝色虚线是 95% 置信区间（约 ±1.96/√n）
- 如果大部分滞后的 ACF 值落在虚线以内，说明自相关不显著
- 如果有明显的 ACF 超出虚线，说明该滞后存在显著自相关

## Ljung-Box 检验

ACF 图是视觉判断，Ljung-Box 检验给出统计显著性：

```r
# 联合检验前 10 阶自相关是否为零
Box.test(index$ret, lag = 10, type = "Ljung-Box")
```

- **H₀**：前 k 阶自相关均为零（市场弱式有效）
- **H₁**：至少存在一阶显著的自相关

如果 p 值 > 0.05，不能拒绝原假设，数据支持弱式有效。

也可以逐阶检验：

```r
for (k in 1:10) {
  bt <- Box.test(index$ret, lag = k, type = "Ljung-Box")
  cat(sprintf("lag=%2d  statistic=%.3f  p-value=%.3f\n", k, bt$statistic, bt$p.value))
}
```

## 常见结果

- 成熟市场（美股）通常表现为弱式有效，自相关不显著
- A 股市场有时会在短期内出现微弱自相关（散户交易行为导致），但幅度通常很小
- 换不同的指数（上证综指 000001）、不同的时间段，结果可能有差异

## 关键点

- 弱式有效**不要求**每个自相关系数精确为零，统计上不显著即可
- Ljung-Box 检验的 `lag` 参数影响结论，一般取 5、10、20 做对比
- 可以对比不同时间段的结果（例如 2015 年 vs 2023 年），观察市场效率是否在改善
