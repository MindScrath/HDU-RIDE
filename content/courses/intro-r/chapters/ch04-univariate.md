# 第4章 一元时间序列分析

在前三章中，我们建立起了时间序列分析的理论框架，推导了 ARMA 模型的优美数学性质。从本章开始，我们将走出纯粹的理论推导，进入实战环节：面对一条真实存在的金融时间序列，我们该如何检验它是否平稳？如果平稳，如何用数据去“估计”模型的参数？模型建立后，如何对其进行检验，并最终利用它去预测未来？本章将为您提供一套完整的一元时间序列分析工作流。

## 4.1 平稳性检验

在第2章中，我们通过“看图”来直观感受平稳性。但这远远不够严谨。金融计量学家们发明了极其严格的假设检验方法，来判定一条序列是否存在导致非平稳的“单位根”（Unit Root）。

### 4.1.1 DF 检验 (Dickey-Fuller Test)

考虑最简单的一阶自回归过程 AR(1)：
$$y_t = \phi y_{t-1} + \varepsilon_t$$
我们知道，当 $\phi = 1$ 时，它就变成了非平稳的随机游走过程（此时特征方程的根为 $\lambda = 1$，这就是所谓的“单位根”）。
因此，检验平稳性，本质上就是检验系数 $\phi$ 是否等于 1。
- 原假设 $H_0$：$\phi = 1$ （序列含有单位根，是非平稳的）
- 备择假设 $H_1$：$|\phi| < 1$ （序列是平稳的）

为了方便构建检验统计量，Dickey 和 Fuller 将原方程两边同时减去 $y_{t-1}$，进行如下巧妙的数学变形：
$$\Delta y_t = y_t - y_{t-1} = \phi y_{t-1} - y_{t-1} + \varepsilon_t = (\phi - 1)y_{t-1} + \varepsilon_t$$
令 $\gamma = \phi - 1$，则方程变为：
$$\Delta y_t = \gamma y_{t-1} + \varepsilon_t$$
此时，检验的假设等价转化为：
- 原假设 $H_0$：$\gamma = 0$ （即 $\phi = 1$，存在单位根，非平稳）
- 备择假设 $H_1$：$\gamma < 0$ （即 $\phi < 1$，平稳）

由于在原假设下（含单位根），序列本身就是非平稳的，常规的 t 分布在这里不再适用。Dickey 和 Fuller 通过复杂的蒙特卡洛模拟得出了该统计量在原假设下的极限分布，这就是著名的 **DF 分布**。

### 4.1.2 ADF 检验 (Augmented Dickey-Fuller Test)

基础的 DF 检验有一个致命的数学前提假设：误差项 $\varepsilon_t$ 必须是纯粹的白噪声。然而在现实的金融数据中，短期的价格波动往往自身存在自相关性。如果强行使用原始的 DF 检验，会导致严重的统计偏差。
为了解决这个问题，计量学家提出了**扩展的 DF 检验（ADF 检验）**。其核心思想是在 DF 检验的回归方程右侧，人为添加被解释变量（$\Delta y_t$）的若干滞后项，以“吸收”掉误差项中的自相关性：
$$\Delta y_t = \gamma y_{t-1} + \sum_{i=1}^p \beta_i \Delta y_{t-i} + \varepsilon_t$$
加入了这些差分滞后项之后，残差 $\varepsilon_t$ 就重新被“洗刷”成了干净的白噪声。此时我们对 $\gamma$ 依然进行与 DF 检验完全相同的假设检验（即检验 $\gamma$ 是否显著为 0）。
*注：在实际的软件操作中，回归方程还可以选择是否加入常数项（漂移项）和时间趋势项，这是由该数据原本的走势特征决定的。*

### 4.1.3 单整序列 (Integrated Series) 与最佳差分次数

如果一条时间序列本身是不平稳的（即未通过 ADF 检验，接受了原假设），但对其进行 $d$ 次差分后变成了平稳序列，我们就称原序列为 **$d$ 阶单整序列**，记作 **$I(d)$**。
- 平稳序列本身不需要任何差分，记为 $I(0)$。
- 股票价格通常是 $I(1)$ 的，因为其一阶差分（即收益率序列）通常是平稳的。

**如何判断合适的差分次数？**
在实战中，我们遵循以下严谨的步骤：
1. 对原序列进行 ADF 检验，如果 p-value < 0.05（拒绝原假设），说明序列已经平稳，停止差分（阶数为 0）。
2. 如果 p-value > 0.05（不平稳），则对序列进行一阶差分，并对差分后的新序列再次进行 ADF 检验。如果平稳，则阶数 $d = 1$。
3. 依此类推。
> [!WARNING]
> 金融数据极少出现需要差分 2 次及以上的情况！如果您发现您的数据需要二阶甚至三阶差分才能通过 ADF 检验，请务必警惕我们在第二章讲过的**过差分（Over-differencing）**问题，您的操作可能反而放大了数据噪声。

## 4.2 模型的估计

当我们确认序列已经平稳（或者通过差分使其平稳），并通过理论判断初步确定了 ARMA 模型的 $p$ 和 $q$ 阶数后，下一步就是交给计算机去“估计”方程中的参数（即计算出 $\phi$ 和 $\theta$ 的具体数值）。

### 4.2.1 矩估计 (Method of Moments)

矩估计的哲学思想非常朴素：用样本计算出的统计量（如样本均值、样本自相关系数）去“强行等于”理论模型中的数学公式。
以 AR(1) 为例，在第3章中我们推导出其理论一阶自相关系数为 $\rho_1 = \phi$。
在拿到真实的样本数据后，我们可以直接算出一个确定的样本一阶自相关系数 $\hat{\rho}_1$。那么，根据矩估计的逻辑，参数 $\phi$ 的估计值直接就是：
$$\hat{\phi} = \hat{\rho}_1$$
矩估计的计算极其简单迅速，但由于它没有充分利用样本数据分布中的全部特征信息，其估计精度（尤其是在处理包含移动平均 MA 的模型时）通常是较差的。在现代金融软件中，矩估计经常仅被用来作为更高级算法的“初始迭代值”。

### 4.2.2 最小二乘估计 (OLS)

对于**纯粹的自回归 AR 模型**，普通的最小二乘法（Ordinary Least Squares, OLS）是非常优秀的估计方法。
考虑 AR(1) 模型：$y_t = c + \phi y_{t-1} + \varepsilon_t$
您会惊奇地发现，这与我们在基础统计学中学过的一元线性回归方程 $Y = \beta_0 + \beta_1 X + u$ 在代数形式上完全一致！我们只需将当期的 $y_t$ 作为被解释变量，滞后一期的 $y_{t-1}$ 作为解释变量，寻找使残差平方和（即 $\sum \varepsilon_t^2$）达到最小的 $c$ 和 $\phi$ 即可。
理论证明，OLS 估计在 AR 模型中具有非常良好的统计一致性和渐近有效性。

### 4.2.3 极大似然估计 (MLE)

极大似然估计（Maximum Likelihood Estimation, MLE）是现代金融计量学中最主流、也是最强大的参数估计方法，尤其在处理包含 MA 项的复杂 ARMA 模型时，它是不可或缺的。
它的核心逆向思维是：既然现实中我们已经确切地观察到了这组唯一的样本数据，那么我们就去寻找一组完美的参数，**使得在理论上产生当前这组样本数据的总体概率（即似然函数）达到最大**。
在时间序列分析中，我们通常假设底层的白噪声 $\varepsilon_t$ 服从正态分布。基于此，我们可以写出整个历史序列联合分布的对数似然函数。通过复杂的非线性优化算法（如牛顿-拉夫逊法），计算机能在极短时间内精确地找到使似然函数最大的那组参数。

## 4.3 模型的推断与体检

计算机火速给出了参数估计值，并不意味着建模工作就结束了。我们必须对模型进行严苛的“体检”，确保模型已经榨干了数据中所有有用的规律。

### 4.3.1 残差检验

检验一个时间序列模型好坏的最核心标准只有一个：**看残差**。
如果模型已经完美捕捉了序列的内在记忆规律，那么剥离掉这些规律后，剩下的残差 $\hat{\varepsilon}_t$ 应该只包含完全无法预测的纯随机噪声。换句话说，**模型的残差序列必须是一个白噪声。**

具体做法是：将模型拟合出的残差序列提取出来，进行第2章中讲过的 **Ljung-Box (LB) 检验**。
- 如果 LB 检验拒绝了原假设（即 p-value < 0.05），说明残差中仍残留有自相关性，这意味着当前模型设定有误（滞后阶数不够），没有提炼干净数据里的规律。
- 如果接受了原假设（p-value > 0.05），说明残差已经是干净的白噪声，模型顺利通过体检！

### 4.3.2 滞后阶数的判断与自动选择

在实际操作中，新手面临的最大痛苦是：面对波动无常的金融数据，我到底该给 AR 模型和 MA 模型分别设定多少阶的 $p$ 和 $q$？

**(1) 考察 ACF 和 PACF (传统看图法)**
结合第3章的理论，我们通过肉眼观察 ACF / PACF 图形的“截尾性”来定阶。但在真实且充满噪音的金融数据中，图形往往参差不齐，极难看出教科书般的“突然截断”，这使得看图定阶带有浓厚的主观色彩。

**(2) 信息准则 (AIC / BIC) (现代主流自动选择法)**
现代计量经济学信奉“奥卡姆剃刀”原则（如无必要，勿增实体）。我们希望模型能尽量精确地拟合数据，同时又希望模型尽量简单（防止参数过多导致过拟合）。
- **赤池信息准则 (AIC)**：$AIC = -2 \ln(L) + 2k$ （其中 $L$ 为似然函数最大值，$k$ 为模型参数个数）
- **贝叶斯信息准则 (BIC)**：$BIC = -2 \ln(L) + k \ln(T)$ （其中 $T$ 为样本量。由于多乘了 $\ln(T)$，BIC 对复杂臃肿的模型惩罚更重）
在实战中，我们让计算机穷举尝试拟合无数种不同的组合（如 ARMA(1,1), ARMA(2,1), ARMA(3,2) 等），然后挑选出 **AIC 或 BIC 值最小**的那个模型，作为全局最优模型。

## 4.4 模型的预测

千辛万苦建立平稳的时间序列模型，终极目的都是为了预知未来。预测分为“条件预测”（基于当前已知的所有市场信息去推测下一步）和“非条件预测”（站在无穷久远的过去去盲猜）。

### 4.4.1 对 AR(1) 的预测

假设我们身处 $t$ 时刻，已经确切知道了历史行情 $y_t, y_{t-1}, \dots$，想要预测未来的 $y_{t+1}, y_{t+2}, \dots$。已知真实的演化模型是 $y_{t+k} - \mu = \phi(y_{t+k-1} - \mu) + \varepsilon_{t+k}$。

**(1) 条件预测**
站在 $t$ 时刻，预测 $t+1$ 时刻的值（我们将预测值记为 $\hat{y}_{t+1|t}$）：
$$\hat{y}_{t+1|t} = E(y_{t+1} | \text{已知信息}) = \mu + \phi(y_t - \mu) + E(\varepsilon_{t+1}) = \mu + \phi(y_t - \mu)$$
（因为对于未来时刻 $t+1$ 发生的白噪声冲击，我们在 $t$ 时刻预期它为平均值 0）。

进一步，预测 $t+2$ 时刻的值：
$$\hat{y}_{t+2|t} = \mu + \phi(\hat{y}_{t+1|t} - \mu) = \mu + \phi^2(y_t - \mu)$$
以此类推，向前 $k$ 步的条件预测普遍公式为：
$$\hat{y}_{t+k|t} = \mu + \phi^k(y_t - \mu)$$

**(2) 均值回复 (Mean Reversion) 与非条件预测**
深刻观察上面的向前 $k$ 步预测公式，由于 AR(1) 是平稳的，其平稳条件为 $|\phi| < 1$。
当我们把预测的期限拉得无限长，即预测步数 $k \to \infty$ 时，指数项 $\phi^k$ 将衰减至 0。因此：
$$\lim_{k \to \infty} \hat{y}_{t+k|t} = \mu$$
这揭示了金融市场中一个极其深刻的哲学规律：**均值回复（Mean Reversion）**。无论当前的资产价格或收益率偏离其长期均值 $\mu$ 有多远，由于内在引力的存在，随着时间的推移，我们对其遥远未来的预测终将逐渐收敛、回归到它的长期均值。
而在没有任何已知历史信息的条件下进行“非条件预测”，其预测结果也就是这个常数序列的数学期望 $\mu$。

### 4.4.2 对 MA(1) 的预测

对于移动平均 MA(1) 模型：$y_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1}$

**(1) 条件预测**
向前 1 步预测（在 $t$ 时刻，$t$ 时刻的白噪声冲击 $\varepsilon_t$ 已经是可通过计算得知的残差）：
$$\hat{y}_{t+1|t} = \mu + E(\varepsilon_{t+1}) + \theta_1 \varepsilon_t = \mu + \theta_1 \varepsilon_t$$
向前 2 步预测：
$$\hat{y}_{t+2|t} = \mu + E(\varepsilon_{t+2}) + \theta_1 E(\varepsilon_{t+1}) = \mu + 0 + 0 = \mu$$
由于 MA(1) 只拥有一阶的历史记忆，当我们的预测步数 $k \ge 2$ 时，所有基于当前最新冲击的影响会瞬间蒸发，预测值立刻僵硬地回归到了长期均值 $\mu$。

**(2) 比较与优劣**
MA 模型的预测“短视”特征非常明显，它只能对极其短期的未来进行有效预测；而 AR 模型由于其指数“拖尾”的记忆性，预测能力能够辐射得更深远，尽管这种预测效力也会随着时间逐渐衰减。

### 4.4.3 对 ARMA(1,1) 的预测

对于综合模型：$y_{t+1} - \mu = \phi(y_t - \mu) + \varepsilon_{t+1} + \theta \varepsilon_t$

**(1) 条件预测**
向前 1 步：
$$\hat{y}_{t+1|t} = \mu + \phi(y_t - \mu) + \theta \varepsilon_t$$
（这非常聪明：它既利用了上一期的系统状态/价格，又提取了上一期系统残余的白噪声冲击信息）
向前 2 步：
$$\hat{y}_{t+2|t} = \mu + \phi(\hat{y}_{t+1|t} - \mu) + \theta E(\varepsilon_{t+1}) = \mu + \phi(\hat{y}_{t+1|t} - \mu)$$
从第 2 步开始，MA 部分的短期冲击红利消耗殆尽，ARMA 模型的长线预测行为完美退化为了纯粹的 AR 模型！此后，预测值将以系数 $\phi$ 为速率，平滑地发生“均值回复”。

### 💻 实战代码：R 语言全流程自动分析演示

说了这么多理论，在真正的现代工作中，我们是否需要手算推导去检验、定阶呢？不需要！
我们使用 R 语言中代表最先进生产力的 `fable` 体系（结合我们之前用过的 `tsibble`），对一条带有趋势的非平稳序列进行“平稳性检验 $\to$ 自动定阶与估计 $\to$ 残差体检 $\to$ 预测制图”的傻瓜式全流程演示。

```R
# 载入核心依赖包
library(tidyverse)
library(tsibble)
library(feasts)
library(fable)
library(tseries) # 用于执行经典的 ADF 检验

# 设定随机种子
set.seed(999)
# 我们让系统后台悄悄模拟生成一条带趋势的 ARIMA(1,1,1) 序列数据，总长度 150
sim_data <- arima.sim(model = list(order = c(1,1,1), ar = 0.6, ma = 0.4), n = 150)
df_ts <- tibble(date = 1:151, value = as.numeric(sim_data)) %>% as_tsibble(index = date)

# --------------------------
# 步骤 1：平稳性初步检验 (ADF 检验)
# --------------------------
# 对原始观测序列进行检验 (原假设：非平稳)
adf.test(df_ts$value)
# 如果发现 p-value > 0.05，说明数据非平稳，通常需要进行差分。

# --------------------------
# 步骤 2：模型估计与完美自动定阶 (信息准则法)
# --------------------------
# 使用 fable 体系下的极简 ARIMA 函数
fit_model <- df_ts %>%
  model(
    # 这行代码极为强大：它会自动尝试各种差分次数和 p,q 组合，
    # 并根据 AIC 准则在后台选出全局最优模型进行 MLE 估计！
    auto_arima = ARIMA(value) 
  )

# 查看计算机最终拍板的最优模型参数
report(fit_model) 

# --------------------------
# 步骤 3：对模型进行“残差体检”
# --------------------------
# 极简一键生成残差的分布图和 ACF 图
fit_model %>% gg_tsresiduals()

# 进行严密的 Ljung-Box 白噪声检验 (设置滞后 10 期)
augment(fit_model) %>%
  features(.innov, ljung_box, lag = 10)
# 若输出框的 p_value 稳稳大于 0.05，恭喜，残差已是白噪声，模型提取干净了！

# --------------------------
# 步骤 4：向未来预测并画图
# --------------------------
# 勇敢地向前预测未来 20 期行情
fc <- fit_model %>% forecast(h = 20)

# 绘制原数据和优美的预测扇形区（包含 80% 和 95% 置信区间边界）
fc %>%
  autoplot(df_ts, color = "darkblue") +
  labs(title = "基于最优 ARIMA 模型的一元时间序列预测", x = "时间", y = "数值") +
  theme_bw()
```
> [!TIP]
> 结合上述代码你会发现：在这套极其现代化的 R 语言工作流中，痛苦的主观定阶工作被 `ARIMA()` 函数底层的 AIC 优化算法完美替代了。作为金融分析师，我们终于可以从繁琐的数学计算中解放出来，将核心精力集中在解读经济学意义、严抓残差体检（防范模型失效风险）以及对预测结果的业务洞察上。
