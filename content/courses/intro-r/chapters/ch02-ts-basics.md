# 第2章 时间序列基础

在上一章中，我们认识了什么是金融计量学，并掌握了对数收益率的计算。如果我们把一只股票每天的对数收益率按时间顺序排列起来，就构成了一个“时间序列”。本章是理解后续所有高级金融计量模型的基础。我们将从最基本的概念出发，逐步揭开时间序列的神秘面纱。

## 2.1 时间序列与随机过程

### 2.1.1 随机过程的定义

在概率论中，我们熟悉“随机变量”的概念，比如抛一次硬币的结果。如果我们将时间因素引入进来，把一连串按时间排序的随机变量集合在一起，就构成了一个**随机过程**（Stochastic Process）。
假设 $T$ 是一个时间指标的集合（比如 $t=1, 2, 3, \cdots$），对于每一个 $t \in T$，都有一个随机变量 $Y_t$。那么，集合 $\{Y_t, t \in T\}$ 就是一个随机过程。

**时间序列（Time Series）** 其实就是随机过程的一次具体“实现”（Realization）。换句话说，随机过程是理论上的模型，而我们手中拿到的一组真实历史数据（比如过去100天的股票收盘价），只是这个理论模型生成的一个样本。

### 2.1.2 统计特征：期望、方差、自协方差与自相关函数

为了刻画一个时间序列，我们需要了解它在每个时间点的状态以及不同时间点之间的关联。

**(1) 期望（Mean）**
时间序列在时刻 $t$ 的期望（均值）反映了该时刻变量的中心位置，记为 $\mu_t$：
$$\mu_t = E(Y_t)$$

**(2) 方差（Variance）**
方差衡量了时间序列在时刻 $t$ 偏离其均值的程度，即波动大小：
$$Var(Y_t) = E[(Y_t - \mu_t)^2]$$

**(3) 自协方差（Autocovariance）**
在经典的统计学中，协方差用来衡量两个不同变量 $X$ 和 $Y$ 之间的线性相关性。在时间序列中，我们只有一个变量 $Y$，但我们可以衡量它在不同时间点（比如 $Y_t$ 和 $Y_{t-k}$）之间的相关性。因为是跟自己历史上的值算协方差，所以称为**自协方差**。
定义时间序列在时刻 $t$ 和时刻 $t-k$ 的自协方差为 $\gamma_{t, t-k}$：
$$\gamma_{t, t-k} = Cov(Y_t, Y_{t-k}) = E[(Y_t - \mu_t)(Y_{t-k} - \mu_{t-k})]$$
其中，$k$ 称为滞后期数（Lag）。

**(4) 自相关函数（Autocorrelation Function, ACF）**
自协方差的值依赖于变量的量纲，为了消除量纲的影响，我们将其标准化，得到**自相关系数**。时刻 $t$ 和 $t-k$ 之间的自相关系数定义为：
$$\rho_{t, t-k} = \frac{Cov(Y_t, Y_{t-k})}{\sqrt{Var(Y_t)} \sqrt{Var(Y_{t-k})}} = \frac{\gamma_{t, t-k}}{\sqrt{\gamma_{t, t}} \sqrt{\gamma_{t-k, t-k}}}$$
所有滞后 $k$ 的自相关系数构成了一个函数，这就是我们在时间序列分析中最常用的工具之一——**自相关函数（ACF）**。

## 2.2 平稳性

如果我们不对时间序列加以任何限制，那么它的期望、方差和自相关系数都可能随着时间 $t$ 剧烈变化。这样一来，我们无法利用过去的数据去预测未来，因为未来的“游戏规则”完全变了。因此，我们需要引入“平稳性”（Stationarity）的概念。

### 2.2.1 强平稳（Strict Stationarity）与弱平稳（Weak Stationarity）

**(1) 强平稳**
强平稳的定义非常严苛：如果对于任意时间点 $t_1, t_2, \cdots, t_n$ 和任意时间间隔 $k$，联合概率分布 $F(Y_{t_1}, Y_{t_2}, \cdots, Y_{t_n})$ 与 $F(Y_{t_1+k}, Y_{t_2+k}, \cdots, Y_{t_n+k})$ 完全相同，那么该序列就是强平稳的。
这要求序列的整个概率分布特征不随时间推移而改变。在现实的金融数据中，强平稳几乎不可能满足。

**(2) 弱平稳（协方差平稳）**
由于强平稳太难实现，我们在金融计量学中通常使用的是**弱平稳**的概念。弱平稳只要求序列的前两阶矩（即均值和方差/协方差）保持稳定。一个时间序列 $\{Y_t\}$ 是弱平稳的，必须同时满足以下三个条件：
1. **均值是常数**：$E(Y_t) = \mu$，与时间 $t$ 无关。
2. **方差是常数**：$Var(Y_t) = \gamma_0$，与时间 $t$ 无关。
3. **自协方差仅依赖于滞后期数 $k$**，而与时间 $t$ 无关：$Cov(Y_t, Y_{t-k}) = \gamma_k$。

对于弱平稳时间序列，由于方差不随时间改变，且 $\gamma_0 = Var(Y_t)$，它的自相关函数（ACF）可以大大简化为：
$$\rho_k = \frac{\gamma_k}{\gamma_0}$$
本书后续提到的“平稳”，除非特殊说明，均指代**弱平稳**。

### 2.2.2 遍历定理（Ergodic Theorem）

即使序列是平稳的，我们依然面临一个现实困境：在真实世界中，金融资产的时间序列只发生过一次（比如昨天的茅台股价只有一个，不可能重新上演）。我们只有时间维度的**一条**样本路径。
**遍历性**（Ergodicity）回答了这样一个问题：我们能否用时间维度的平均值，去替代概率空间上的期望值（集合平均）？
简单来说，如果一个平稳时间序列满足遍历定理，那么随着样本数量 $T \to \infty$，样本的时间平均值（比如样本均值 $\frac{1}{T} \sum_{t=1}^T Y_t$）就会依概率收敛于真实的理论期望值 $\mu$。
遍历定理赋予了我们在现实世界中利用有限历史数据去估计总体统计量的合法性。

## 2.3 白噪声 (White Noise)

在平稳序列中，最简单、最基础的模型就是白噪声。它就像是时间序列分析中的“原子”，很多复杂的模型都是由白噪声构建而成的。

### 2.3.1 白噪声的定义及其统计特征

如果一个时间序列 $\{\varepsilon_t\}$ 满足以下三个条件，我们就称其为**白噪声过程**（White Noise Process）：
1. **均值为 0**：$E(\varepsilon_t) = 0$
2. **方差为常数**：$Var(\varepsilon_t) = \sigma^2$
3. **序列不存在自相关**：对于任意 $k \neq 0$，协方差 $Cov(\varepsilon_t, \varepsilon_{t-k}) = 0$

通过上述定义，我们可以立即写出白噪声的自相关函数（ACF）：
$$
\rho_k = 
\begin{cases}
1, & k = 0 \\
0, & k \neq 0
\end{cases}
$$
这意味着，白噪声的过去、现在和未来是完全独立、互不影响的。它的走势纯粹是随机的，你无法从过去的数据中预测未来的任何信息。

### 2.3.2 白噪声与弱平稳之间的关系

对照弱平稳的三个条件，我们可以清楚地发现：白噪声完美满足均值为常数（0）、方差为常数（$\sigma^2$）、且自协方差仅依赖于时间间隔 $k$（当 $k=0$ 时为 $\sigma^2$，当 $k \neq 0$ 时为 0）。
因此，**白噪声是一个特殊的弱平稳过程**。

## 2.4 随机游走过程 (Random Walk)

说完了最简单的平稳序列（白噪声），我们来看一个金融界最著名的**非平稳序列**——随机游走。“随机游走”假说曾深刻影响了整个华尔街，它认为股票价格的变动如同醉汉走路，完全不可预测。

### 2.4.1 随机游走过程的构建

随机游走过程可以视为过去所有白噪声的简单求和。设 $\{\varepsilon_t\}$ 为均值为 0、方差为 $\sigma^2$ 的白噪声序列，随机游走 $\{Y_t\}$ 定义为：
$$Y_t = Y_{t-1} + \varepsilon_t$$
这说明，今天的价格等于昨天的价格加上一个完全随机的白噪声冲击。为了便于分析，假设初始状态 $Y_0 = 0$，我们可以通过迭代法将 $Y_t$ 展开：
$$Y_1 &= Y_0 + \varepsilon_1 = \varepsilon_1 \\
Y_2 &= Y_1 + \varepsilon_2 = \varepsilon_1 + \varepsilon_2 \\
&\vdots \\
Y_t &= \varepsilon_1 + \varepsilon_2 + \cdots + \varepsilon_t = \sum_{i=1}^t \varepsilon_i$$
可见，随机游走确实是白噪声的累积求和过程。

### 2.4.2 随机游走的统计特征

由于 $Y_t = \sum_{i=1}^t \varepsilon_i$，我们来推导它的均值、方差和自相关函数。

**(1) 均值**
因为白噪声的均值为0：
$$E(Y_t) = E\left(\sum_{i=1}^t \varepsilon_i\right) = \sum_{i=1}^t E(\varepsilon_i) = 0$$

**(2) 方差**
由于白噪声之间互相独立：
$$Var(Y_t) = Var\left(\sum_{i=1}^t \varepsilon_i\right) = \sum_{i=1}^t Var(\varepsilon_i) = t \sigma^2$$
**这是一个极其重要的结论！** 随机游走的方差是随时间 $t$ 线性增长的。当 $t \to \infty$ 时，方差趋于无穷大。由于方差不是常数，这直接证明了**随机游走是一个非平稳过程**。

**(3) 自协方差与自相关函数**
对于滞后期数 $k > 0$：
$$Cov(Y_t, Y_{t-k}) &= Cov(Y_{t-k} + \varepsilon_{t-k+1} + \cdots + \varepsilon_t, Y_{t-k})$$
因为后期的白噪声与前期的状态无关（即 $Cov(\varepsilon_i, Y_{t-k}) = 0, \forall i > t-k$），所以：
$$Cov(Y_t, Y_{t-k}) = Cov(Y_{t-k}, Y_{t-k}) = Var(Y_{t-k}) = (t-k)\sigma^2$$

自相关函数为：
$$\rho_{t, t-k} &= \frac{Cov(Y_t, Y_{t-k})}{\sqrt{Var(Y_t)} \sqrt{Var(Y_{t-k})}} \\
&= \frac{(t-k)\sigma^2}{\sqrt{t\sigma^2} \sqrt{(t-k)\sigma^2}} \\
&= \sqrt{\frac{t-k}{t}} = \sqrt{1 - \frac{k}{t}}$$
可以看出，当 $t$ 很大而 $k$ 相对较小时，$\rho_{t, t-k} \approx 1$。这说明随机游走序列的记忆性非常强，当期状态与前几期状态高度相关，且其相关性随时间推移衰减得极其缓慢。

### 2.4.3 差分处理与过差分问题 (含R代码演示)

对于非平稳的随机游走过程，我们通常使用**差分（Difference）**操作将其转换为平稳过程。一阶差分定义为 $\Delta Y_t = Y_t - Y_{t-1}$。
显然，对于随机游走模型 $Y_t = Y_{t-1} + \varepsilon_t$，我们有一阶差分：
$$\Delta Y_t = Y_t - Y_{t-1} = \varepsilon_t$$
通过简单的一阶差分，非平稳的随机游走瞬间变成了完美的平稳序列（白噪声），其均值为 0，方差为稳定的 $\sigma^2$。

但是，差分次数并不是越多越好。如果我们对已经平稳的白噪声再进行一次差分（即对原序列进行二阶差分）：
$$\Delta^2 Y_t = \Delta Y_t - \Delta Y_{t-1} = \varepsilon_t - \varepsilon_{t-1}$$
我们来计算二阶差分后的方差：
$$Var(\Delta^2 Y_t) &= Var(\varepsilon_t - \varepsilon_{t-1}) \\
&= Var(\varepsilon_t) + Var(\varepsilon_{t-1}) \\
&= 2\sigma^2$$
我们发现，方差不仅没有变小，反而变大了一倍！这就是所谓的**过差分（Over-differencing）**问题。过差分不仅会放大系统的噪声方差，还会人为引入不必要的负相关性，增加后续建模的难度。

我们可以通过 R 语言来进行具体的模拟和比较。

```R
# 载入数据处理和可视化所需的包
library(tidyverse)
library(tsibble)
library(cowplot)

# 设定随机种子以保证每次运行结果可复现
set.seed(123)

# 1. 模拟随机游走过程
n <- 500
white_noise <- rnorm(n, mean = 0, sd = 1) # 生成500个标准正态白噪声
random_walk <- cumsum(white_noise)        # 累加得到随机游走序列

# 转换为 tsibble 数据结构方便后续处理和画图
df <- tibble(
  time = 1:n,
  Y = random_walk,
  # 2. 一阶差分
  diff_1 = Y - lag(Y),
  # 3. 二阶差分
  diff_2 = diff_1 - lag(diff_1)
) %>%
  as_tsibble(index = time)

# 绘制三张图进行直观对比
# 图1：原序列
p1 <- df %>% ggplot(aes(x = time, y = Y)) + 
  geom_line(color = "darkred") + 
  labs(title = "原序列：随机游走 (非平稳，表现出明显趋势)", x = "时间", y = "Y") + 
  theme_bw()

# 图2：一阶差分
p2 <- df %>% ggplot(aes(x = time, y = diff_1)) + 
  geom_line(color = "steelblue") + 
  labs(title = "一阶差分：白噪声 (平稳，围绕0均匀波动)", x = "时间", y = expression(Delta*Y)) + 
  theme_bw()

# 图3：二阶差分
p3 <- df %>% ggplot(aes(x = time, y = diff_2)) + 
  geom_line(color = "darkgreen", alpha = 0.7) + 
  labs(title = "二阶差分：过差分 (方差明显被放大)", x = "时间", y = expression(Delta^2*Y)) + 
  theme_bw()

# 将三张图纵向排列在一起展示
plot_grid(p1, p2, p3, nrow = 3, align = 'v')
```
*注：运行上述代码后，你将直观地看到：原序列有很强的不可预测趋势；一阶差分后变成了围绕 0 均匀波动的平稳白噪声；而二阶差分后的图形变得异常密集，其波动的振幅（方差）在视觉上明显大于一阶差分。*

## 2.5 样本统计量与自相关图绘制

在现实世界中，我们永远无法得知金融数据的“真实”生成过程（比如真实的理论均值 $\mu$ 或真实的自协方差 $\gamma_k$），我们只能用手中有限的历史样本去“估计”它们。

### 2.5.1 样本统计量公式

假设我们拥有 $T$ 个真实的样本观测值 $y_1, y_2, \cdots, y_T$。

**(1) 样本均值 $\bar{y}$**
$$\bar{y} = \frac{1}{T} \sum_{t=1}^T y_t$$

**(2) 样本方差 $\hat{\gamma}_0$**
$$\hat{\gamma}_0 = \frac{1}{T-1} \sum_{t=1}^T (y_t - \bar{y})^2$$

**(3) 样本自协方差 $\hat{\gamma}_k$**
$$\hat{\gamma}_k = \frac{1}{T} \sum_{t=k+1}^T (y_t - \bar{y})(y_{t-k} - \bar{y})$$
*注意：在计算时间序列样本自协方差时，分母通常习惯除以 $T$ 而不是 $(T-k)$，这在数学上可以保证最终生成的自协方差矩阵是半正定的。*

**(4) 样本自相关函数 (Sample ACF) $\hat{\rho}_k$**
$$\hat{\rho}_k = \frac{\hat{\gamma}_k}{\hat{\gamma}_0}$$

### 2.5.2 在R中绘制ACF图

样本自相关图是判断时间序列平稳性和选择模型的“眼睛”。我们利用 R 语言中强大的作图包来绘制上一节生成的白噪声（一阶差分序列）的 ACF 图。

```R
library(feasts) # feasts 包是 tsibble 体系下用于时间序列特征提取和绘图的极佳工具

# 提取一阶差分序列，并去除第一行因 lag 产生的 NA 值
df_diff1 <- df %>% filter(!is.na(diff_1))

# 绘制 ACF 图
df_diff1 %>%
  # 计算最大滞后20期的ACF
  ACF(diff_1, lag_max = 20) %>%
  ggplot(aes(x = lag, y = acf)) +
  # 画出红色的火柴棍形状代表自相关系数大小
  geom_segment(aes(x = lag, xend = lag, y = 0, yend = acf), color = 'red', linewidth = 1) +
  geom_point(shape = 21, color = 'red', fill = 'red', size = 2) +
  # 添加 95% 置信区间虚线
  geom_hline(yintercept = c(1.96/sqrt(n), -1.96/sqrt(n)), linetype = "dashed", color = "blue") +
  labs(title = "一阶差分（白噪声）的样本自相关函数 (ACF) 图", x = "滞后期数 (Lag)", y = "自相关系数 ACF") +
  theme_bw()
```
*注：在生成的 ACF 图中，横坐标是滞后期数 $k$，纵坐标是样本自相关系数。蓝色的虚线是 95% 的置信区间带。对于真正的白噪声序列，除了滞后 0 期（自己和自己相关系数永远为 1）之外，绝大多数的红色火柴棍都应该落在蓝色虚线范围内部。*

## 2.6 白噪声检验

“看图”毕竟带有主观性。为了严谨起见，我们需要进行纯统计学上的假设检验，以确定一个序列究竟是不是白噪声。核心思想是检验该序列的前 $m$ 个自相关系数 $\rho_1, \rho_2, \cdots, \rho_m$ 是否“联合”为 0。

- **原假设 $H_0$**：序列是白噪声（即 $\rho_1 = \rho_2 = \cdots = \rho_m = 0$）
- **备择假设 $H_1$**：序列不是白噪声（即至少有一个自相关系数显著不为 0）

### 2.6.1 Box-Pierce Q 统计量

Box 和 Pierce 提出了如下检验统计量：
$$Q = T \sum_{k=1}^m \hat{\rho}_k^2$$
其中 $T$ 是样本容量，$m$ 是我们人为设定的最大滞后期数。如果在原假设成立的前提下，$Q$ 统计量渐近服从自由度为 $m$ 的卡方分布 $\chi^2(m)$。如果计算出的 $Q$ 值大于分布的临界值，则拒绝原假设，认为序列存在显著的自相关性，即它不是白噪声。

### 2.6.2 Ljung-Box 统计量 (LB 统计量)

由于 Q 统计量在小样本（即 $T$ 较小）下的检验表现不佳，Ljung 和 Box 对其进行了修正，提出了目前金融计量中应用最广泛的 **LB 统计量**：
$$Q_{LB} = T(T+2) \sum_{k=1}^m \frac{\hat{\rho}_k^2}{T-k}$$
同样，在原假设下，$Q_{LB} \sim \chi^2(m)$。LB 统计量在小样本和大样本下都具有良好的检验性质，因此在现代软件中几乎都默认使用 LB 检验。

在 R 语言中，进行 LB 检验只需一行代码：
```R
# 对一阶差分序列（白噪声）进行 Ljung-Box 检验，滞后期设定为 10
# 注意：原假设是“该序列为白噪声”
box_test_result <- Box.test(df_diff1$diff_1, lag = 10, type = "Ljung-Box")

# 打印检验结果
print(box_test_result)
```
*注：在 R 语言输出的结果中，重点观察 `p-value`（p 值）。如果 `p-value > 0.05`（即在 5% 的显著性水平下），我们就**不能拒绝原假设**，从而在统计学上确认该序列就是白噪声。这与我们之前观察 ACF 图得到的结论应该是一致的。*

### 小结

本章我们深入探讨了时间序列分析的基石。从认识随机过程、平稳性，到剖析最简单的平稳序列（白噪声）和最著名的非平稳序列（随机游走），我们详细推导了它们的均值、方差和自相关函数。同时，我们掌握了如何用 R 语言通过差分将非平稳转化为平稳序列，并使用 ACF 图和 Ljung-Box 统计量对白噪声进行检验。这一切，都为我们在下一章学习更加复杂的 ARMA 模型做好了完美的理论与代码准备。
