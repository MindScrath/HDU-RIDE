# 第6章 多元时间序列分析

在上一章中，我们讨论了单个方程的时间序列回归模型（如 ARDL），其核心思想是人为地将变量划分为“解释变量 $X$”和“被解释变量 $Y$”。这就如同假定 $X$ 是太阳，$Y$ 是地球，地球只能被动地围绕太阳运转。
然而，在极其复杂的宏观经济和金融网络中，这种单向的因果假设往往过于傲慢。比如，货币政策的宽松会推高股票价格，但股票市场的极度繁荣或崩盘也会反过来迫使央行调整货币政策。变量之间往往是**互为因果、相互决定的**。
为了打破这种“主从关系”的枷锁，我们正式进入**多元时间序列分析**的宇宙——在这里，所有的变量一视同仁，它们共同组成一个庞大的生态系统，在时间的长河中协同演化。

## 6.1 向量自回归模型（VAR）

1980 年，宏观经济学巨头、诺贝尔奖得主克里斯托弗·西姆斯（Christopher Sims）提出了**向量自回归模型（Vector Autoregression, VAR）**。他极具颠覆性地主张：在没有明确的经济学理论指导前，我们不应主观假定谁是谁的原因；相反，我们应该让**所有的变量都平等地作为内生变量**，让它们各自的历史数据去解释系统当前的全部状态。

### 6.1.1 VAR 模型的数学构建

假设我们的经济系统中有两个核心变量：通货膨胀率 $y_{1,t}$ 和 联邦基金利率 $y_{2,t}$。
在一个一阶的向量自回归模型——即 **VAR(1)** 中，每一个变量的当期值，都由它自身的滞后一期，以及系统中**所有其他变量**的滞后一期共同决定：
$$
\begin{cases}
y_{1,t} = c_1 + \phi_{11} y_{1,t-1} + \phi_{12} y_{2,t-1} + \varepsilon_{1,t} \\
y_{2,t} = c_2 + \phi_{21} y_{1,t-1} + \phi_{22} y_{2,t-1} + \varepsilon_{2,t}
\end{cases}
$$
如果我们有 $k$ 个变量，我们自然可以借助线性代数的矩阵工具，极其优雅地将上述方程组打包成一个紧凑的矩阵方程：
$$Y_t = C + \Phi_1 Y_{t-1} + \varepsilon_t$$
其中：
- $Y_t$ 是包含 $k$ 个变量的列向量。
- $C$ 是常数项列向量。
- $\Phi_1$ 是 $k \times k$ 的系数矩阵，它的每一个元素刻画了不同变量间的交叉滞后影响。
- $\varepsilon_t$ 是相互独立的白噪声列向量组合（称为**新息冲击**）。

推广到 $p$ 阶滞后，一般的 **VAR(p)** 模型矩阵表达为：
$$Y_t = C + \Phi_1 Y_{t-1} + \Phi_2 Y_{t-2} + \cdots + \Phi_p Y_{t-p} + \varepsilon_t$$

> [!IMPORTANT]
> **VAR 模型的严苛前提**：正如一元自回归 AR 模型要求数据平稳一样，建立标准 VAR 模型的一个硬性代数前提是：系统中的**每一个变量都必须是平稳的时间序列**（即必须全部通过上一章讲过的 ADF 检验）。如果变量非平稳，必须先进行差分使其平稳，或者使用本章后面讲到的 VECM 模型。

### 6.1.2 VAR 中的格兰杰因果检验

在 VAR 的庞大系统中，变量之间究竟是谁在驱动谁？我们可以将第 5 章学过的**格兰杰因果检验**无缝推广到 VAR 系统中。

在 VAR 框架下，检验“利率 $y_2$ 是否是导致通胀 $y_1$ 的格兰杰原因”，其数学本质就是去检验：**在 $y_1$ 的那个回归方程中，关于 $y_2$ 的所有滞后项系数（即 $\phi_{12}^{(1)}, \phi_{12}^{(2)}, \dots, \phi_{12}^{(p)}$）是否联合为 0。**
如果联合检验的 F 统计量拒绝了系数全为 0 的原假设，就说明 $y_2$ 的历史轨迹确实为预测 $y_1$ 提供了不可替代的信息，从而在统计上确立了 $y_2 \to y_1$ 的格兰杰因果关系。VAR 模型的这一特性，使得它成为了经济学家用来“挖掘数据背后隐藏因果网络”的最强自动化工具。

### 6.1.3 脉冲响应函数 (Impulse Response Function, IRF)

VAR 模型的系数矩阵极其庞大且难以直接解释经济学意义。为了直观理解这个复杂系统，我们再次祭出“脉冲响应”这一法宝。
与一元 AR 过程类似，借助矩阵形式的滞后算子，任何一个平稳的 VAR(p) 过程都可以转化为无穷阶的向量移动平均 VMA($\infty$) 过程：
$$Y_t = \mu + \varepsilon_t + \Psi_1 \varepsilon_{t-1} + \Psi_2 \varepsilon_{t-2} + \cdots$$
在这里，$\Psi_i$ 是一个个巨大的矩阵。它的经济学意义极其震撼：
假设在 $t$ 时刻，央行突然意外加息，导致利率的白噪声项 $\varepsilon_{2,t}$ 产生了一个单位的正向**脉冲冲击**，而其他所有变量的冲击保持为 0。
那么，通过矩阵 $\Psi_1, \Psi_2, \dots$ 的传导链条，这个最初发生在利率上的单一冲击，将会如水波纹一般，在未来第 1 期、第 2 期、第 $n$ 期不可阻挡地蔓延至通货膨胀率、GDP增速等系统内的所有其他变量。将这种波纹的影响轨迹画成折线图，就是多元空间中的**脉冲响应函数图**。

### 6.1.4 方差分解 (Variance Decomposition)

脉冲响应函数告诉了我们“冲击是如何传播的”。而**方差分解**则从另一个上帝视角，回答了“谁才是真正的幕后黑手”。

在预测未来时，预测值必定会产生预测误差（因为未来充满了未知的白噪声冲击 $\varepsilon$）。系统总的预测误差方差，是由不同变量产生的历史冲击共同汇聚而成的。
方差分解技术，就是将某一个变量（比如股市大盘指数）在未来第 $k$ 期的预测误差总方差，**按照百分比精准拆解**给系统中的各个内生变量。
例如，方差分解的结果可能会冷酷地告诉你：在预测未来 10 个月的中国股市波动时，有 60% 的误差来源可以归因于股市自身的惯性冲击，30% 归因于美国降息带来的汇率冲击，而只有 10% 归因于国内货币增发的冲击。这就为我们评估不同政策或外部冲击的“相对重要性”提供了绝对量化的标尺。

## 6.2 结构向量自回归（SVAR）模型

标准的 VAR 模型在数学上极其优美，但它存在一个致命的经济学缺陷：**它是一个“简化式（Reduced Form）”模型。**
仔细观察 VAR 的方程，等式右侧全部都是**滞后项**（昨天的数据）。它强制假设：当期变量之间（今天和今天）是不能相互影响的！
但在现实的高频交易或宏观月度数据中，央行今天上午宣布降息，股市下午就会暴涨。这种在**同一个时间观测期内的“同期（Contemporaneous）”影响**，在标准 VAR 中是被完全抹杀的。

为了找回这种当期的瞬间互动，经济学家引入了**结构向量自回归（Structural VAR, SVAR）模型**。
SVAR 在标准 VAR 的方程左侧，强行乘以了一个代表“同期相互影响网络”的矩阵 $A_0$：
$$A_0 Y_t = C^* + \Phi_1^* Y_{t-1} + \cdots + U_t$$
这里的 $U_t$ 是真正的、具有经济学含义的**“结构性冲击”**（比如纯粹的货币政策冲击、纯粹的供给侧技术冲击）。

由于数学上的自由度限制，SVAR 模型无法直接通过计算机闭着眼睛估计出来。它必须要求经济学家或分析师站出来，**基于深厚的经济学或金融学理论，人为地对矩阵 $A_0$ 施加“约束条件”**（比如强制规定：在同一个月内，股市的波动绝不可能反过来影响 GDP 的公布值，因此矩阵对应位置的系数必须设定为 0）。
SVAR 将冰冷的统计数据与人类经济学智慧完美结合，是目前各国央行制定货币政策时最依赖的核心模型之一。

## 6.3 不平稳的多元时间序列与协整宇宙

前面所有的 VAR 辉煌，都建立在“数据平稳”的沙丘之上。但如果我们要分析的多个变量（比如汇率、黄金价格、石油价格）全都是发散的非平稳 $I(1)$ 序列呢？

在第 5 章中，我们学习了针对两个变量的 EG 两步法协整检验。但如果系统中有 3 个、5 个甚至 10 个非平稳变量，它们之间可能存在错综复杂的**多个协整关系（多根无形的狗绳）**，此时简单的 EG 检验将彻底失效。

### 6.3.1 Johansen 检验

1988年，索伦·约翰森（Søren Johansen）提出了一种基于极大似然估计和矩阵特征值理论的终极协整检验方法——**Johansen 检验**。
它的核心任务是确定一个包含 $k$ 个非平稳变量的高维系统中，究竟存在多少个独立、稳定的**协整向量（Cointegrating Vectors）**，记为 $r$ 个。
- 如果 $r = 0$，说明系统中没有任何一根弹簧，变量们全在毫无交集地乱飞，此时只能全部差分后去做基础的 VAR。
- 如果 $0 < r < k$，说明系统内部存在 $r$ 个长期稳定的均衡关系网！这是最理想的状态。
- 如果 $r = k$，这在数学上等价于所有变量原本就是平稳的，直接做基础 VAR 即可。

Johansen 检验输出两个核心的统计量供我们做假设检验：
1. **迹检验（Trace Test）**：原假设是协整向量的个数小于等于 $r$。
2. **最大特征值检验（Maximum Eigenvalue Test）**：原假设是协整向量的个数正好等于 $r$。

### 6.3.2 向量误差修正模型（VECM）

如果我们通过 Johansen 检验兴奋地发现，这几个不平稳的变量之间确实存在 $r$ 个协整关系。那么，我们不仅不能把它们简单差分掉，反而应该建立比 VAR 更加宏大的终极模型：**向量误差修正模型（Vector Error Correction Model, VECM）**。

VECM 就是第 5 章 ECM 模型在高维矩阵空间的终极形态。它的方程结构如下：
$$\Delta Y_t = \Pi Y_{t-1} + \Gamma_1 \Delta Y_{t-1} + \cdots + \Gamma_{p-1} \Delta Y_{t-p+1} + \varepsilon_t$$
在这个极其霸气的矩阵方程中：
- $\Delta Y_t$ 和右侧的 $\Delta Y_{t-i}$ 代表了整个经济系统的**短期博弈与动态波动矩阵**。
- **$\Pi Y_{t-1}$ 就是整个多元系统的“定海神针”——误差修正项矩阵！** 
矩阵 $\Pi$ 可以被绝妙地分解为两个矩阵的乘积：$\Pi = \alpha \beta^T$。
其中，$\beta$ 矩阵包含了那 $r$ 个维系宇宙长治久安的**长期协整（均衡）方程**；而 $\alpha$ 矩阵则包含了各个变量在偏离长期均衡时，往回修正的**修正系数（力度）**。

VECM 模型既保留了短期的自由搏击，又锁定长期的生死同盟，是目前计量经济学处理宏观非平稳数据的最高统治者。

### 💻 实战代码：R 语言多元 VAR 与协整分析全家桶

在 R 语言中，处理多元时间序列的无可争议的绝对霸主是 **`vars`** 包。我们将使用包内自带的一组真实的加拿大宏观经济数据（包含生产率 `prod`、就业 `e`、失业率 `U`、实际工资 `rw`），来演示一套完整的多元分析军规流程。

```R
# 载入极其强大的多元时间序列包
library(vars)
library(tidyverse)

# 载入包内自带的加拿大真实宏观经济季度数据集（非平稳）
data(Canada)
# 这是一个包含了 prod, e, U, rw 四个变量的多元时间序列矩阵
print(head(Canada))

# --------------------------
# 步骤 1：Johansen 协整检验
# --------------------------
# 假设这四个变量都是 I(1) 的，我们用 Johansen 检验探究它们之间是否存在长期均衡的“狗绳”
# type = "trace" 表示使用迹检验，ecdet = "const" 表示协整方程中包含常数项
jo_test <- ca.jo(Canada, type = "trace", ecdet = "const", K = 2)
summary(jo_test)
# 解读：如果检验结果表明 r>0（通常通过查看统计量是否大于显著性临界值），
# 则说明存在长期的协整关系，我们应该建立 VECM 模型；
# 为了演示 VAR 的完整功能，我们在后续步骤假设我们将数据差分平稳了，或者强行运行 VAR。

# --------------------------
# 步骤 2：VAR 模型的自动定阶与建立
# --------------------------
# VARselect 函数会根据四大信息准则（AIC, HQ, SC, FPE）自动为 VAR 系统挑选最优的滞后阶数 p
var_order <- VARselect(Canada, lag.max = 8, type = "const")
print(var_order$selection) 

# 假设根据 AIC 准则我们选择了滞后 2 阶，正式估计 VAR(2) 模型
var_model <- VAR(Canada, p = 2, type = "const")
# summary(var_model) # (输出报告非常庞大，包含每一个变量的 OLS 回归结果)

# --------------------------
# 步骤 3：格兰杰因果检验 (以失业率 U 是否引起工资 rw 变化为例)
# --------------------------
# 在构建好的多维 VAR 系统中，检验 U 是否是引起系统其他变量变化的 Granger 原因
causality_test <- causality(var_model, cause = "U")
print(causality_test$Granger)
# 若 p-value < 0.05，说明失业率的变动确实是导致其他经济指标变动的原因。

# --------------------------
# 步骤 4：脉冲响应函数 (IRF) 绘图
# --------------------------
# 模拟一次对失业率 "U" 的突发正向冲击，观察它在未来 10 个季度内如何蔓延至实际工资 "rw"
irf_result <- irf(var_model, impulse = "U", response = "rw", n.ahead = 10, boot = TRUE)
plot(irf_result, main = "失业率(U)对实际工资(rw)的脉冲响应")
# 经济学直觉：通常失业率突然上升（冲击），会导致劳动力供大于求，从而在随后的几个季度内压低实际工资（响应值为负）。

# --------------------------
# 步骤 5：方差分解 (FEVD) 绘图
# --------------------------
# 将实际工资 "rw" 未来预测误差的方差，精准拆解给系统内的四个变量
fevd_result <- fevd(var_model, n.ahead = 10)
plot(fevd_result, plot.type = "single", col = c("darkred", "steelblue", "darkgreen", "orange"))
# 读图方法：这张色彩斑斓的面积图直观展示了：随着预测期数(横轴)的拉长，
# 工资波动的误差来源中，自身惯性占了多少百分比，失业率的冲击又慢慢蚕食了多少百分比。
```
> [!TIP]
> 多元时间序列分析由于涉及到庞大的矩阵运算，手工推导几乎是不可能的。现代分析师的核心竞争力，在于能否**基于扎实的经济学和金融学常识**，去正确地解读这些震撼的 IRF 曲线和方差分解色块背后所隐藏的真实市场博弈故事。
